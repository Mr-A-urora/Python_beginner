## 📖 项目简介

这个仓库记录了我在学习 Python 过程中编写的一些基础代码示例与练习。  
主要面向 **零基础、想入门 AI 或编程的初学者**，内容以简单易懂为主，帮助你快速了解 Python 的核心语法与常见用法。

---

## 🧩 项目内容

本项目包含以下部分（持续完善中）：

- Python 基础语法  
- 变量与数据类型  
- 条件语句与循环 
- 列表和元组
- 字典和集合

这些示例代码都相对独立，方便单独运行与学习。

---

## 🛠️ 环境与运行方式

- **Python 版本**：Python 3.x  
- **依赖安装**：无需额外依赖  
- **运行方式**：  
  1. 克隆仓库  
     ```bash
     git clone https://github.com/Mr-A-urora/Python_beginner.git
     cd Python_beginner
     ```
  2. 使用命令行运行任意文件  
     ```bash
     python 文件名.py
     ```
  或者直接在 **VS Code / PyCharm / Jupyter Notebook** 中打开运行。
